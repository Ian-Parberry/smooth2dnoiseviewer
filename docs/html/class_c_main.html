Detailed Description

The interface between I/O from Windows (input from the drop-down menus, output to the client area of the window), the noise generator, and the GDI+ graphics interface.

void CMain::DrawCoords ( )
private

If m_bShowCoords is true, then draw the coordinates of the top left and bottom right of the noise to the corresponding corners of the bitmap. The font family, font, style, and color of the text are hard-coded.
